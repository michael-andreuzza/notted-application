# Assets

Add the following image files to this folder:

- `icon.png` - App icon (1024x1024px)
- `splash-icon.png` - Splash screen icon (200x200px recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

For now, you can create simple placeholder images or use a tool like:
- https://www.canva.com
- https://figma.com
- https://icon.kitchen (for app icons)
