# Notted - Project Progress

## App Features

- Note creation with List and Text modes
- Checklist items with auto-sink for checked items
- Shake to clear checked items (configurable)
- Haptic feedback (configurable)
- Light/Dark/System theme modes
- Multi-language support (English, Spanish, German, Swedish)
- Note templates (built-in + custom for premium)
- Onboarding modal for first launch
- Settings with theme, language, haptics, shake to clear, legal links
- Premium features via in-app purchase

## Premium Features

- Custom note templates (save any note as template)
- All 5 built-in templates
- Floating action button for quick note creation

## UI/UX

- Minimal design with clean backgrounds
- Custom SVG icons
- Segmented control for List/Text tabs
- Premium banner on home screen
- Template picker modal
- Options menu in note view
